#Adapter模式(适配器模式)

在程序世界中经常会存在现有的程序无法直接使用，需要做适当的变换之后才能使用的情况。
这种用于填补“现有的程序”和“所需的程序”之间差异的设计模式就是Adapter模式。

Adapter模式也被程序Wrapper模式，Wrapper有包装器的意思，就像是精美的包装纸将普通的
商品包装成礼物那样，替我们把某样东西包起来，使其能够适用于其他用途的东西就被称为包
装器或是适配器。
